# React
# 目录介绍
* demo文件夹下面是自己学习React写的一些简单组件
* Learn文件夹下面是这段时间学习React的知识的相关综合Demo
* Poss文件夹下面是开始改造大型后台系统代码

#使用技术
* React
* React-redux
* React-router
* Redux
* Antd

#安装使用
* npm install 下载相关插件
* 执行npm run dll 完成后在执行下面的命令 
* npm run dev就可以看到Demo实例了
* npm run Poss就可以看到部分后台代码，正在努力更新中(目前只能看  应用市场后台部分菜单)
